Importaciones básicas

Usa import y from ... import .... Alias con as.

Mantén tus importaciones ordenadas (estándar, terceros, locales) para seguir PEP 8. Los alias evitan conflictos de nombre y clarifican el origen de cada función.

import math as m
from random import randint

print(m.sqrt(9))
print(randint(1, 6))

Ejercicio: usa datetime para imprimir la fecha actual con formato AAAA-MM-DD.

Mostrar solución 
from datetime import date
hoy = date.today()
print(hoy.strftime("%Y-%m-%d"))

Módulos estándar útiles

os, pathlib, sys, json, csv.

Explora la documentación oficial para descubrir utilidades ocultas (p.ej., pathlib.Path.glob, json.loads con object_hook). Evita reinventar la rueda cuando la biblioteca estándar ya lo cubre.

from pathlib import Path
import json

ruta = Path("datos.json")
ruta.write_text(json.dumps({"ok": True}, ensure_ascii=False), encoding="utf-8")
print(json.loads(ruta.read_text(encoding="utf-8")))

Ejercicio: crea un script que liste archivos .py del directorio actual.

Mostrar solución 
from pathlib import Path
for p in Path('.').glob('*.py'):
    print(p)

Ejercicio: instala requests y realiza un GET a un endpoint público.

Mostrar solución
