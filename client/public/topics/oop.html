Clases y objetos

Define tipos propios con atributos y métodos.

En Python todo es objeto; aprovecha métodos dunder como __str__ para personalizar representaciones o __eq__ para comparaciones. Mantén las clases pequeñas y enfocadas.

class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

    def saludar(self):
        return f"Hola, soy {self.nombre}"

ana = Persona("Ana", 20)
print(ana.saludar())

Ejercicio: añade un método es_mayor_de_edad.

Mostrar solución 
class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad
    def es_mayor_de_edad(self):
        return self.edad >= 18

Encapsulamiento y propiedades

Controla el acceso con propiedades.

Las propiedades permiten validar datos al leer/escribir sin romper la API. Usa el prefijo _ para indicar atributos “privados” por convención.

class Cuenta:
    def __init__(self, saldo=0):
        self._saldo = saldo

    @property
    def saldo(self):
        return self._saldo

    def depositar(self, monto):
        if monto < 0:
            raise ValueError("monto negativo")
        self._saldo += monto

Ejercicio: agrega retirar validando que no quede saldo negativo.

Mostrar solución 
def retirar(self, monto):
    if monto < 0: raise ValueError('negativo')
    if monto > self._saldo: raise ValueError('fondos insuficientes')
    self._saldo -= monto

Herencia

Reutiliza y especializa comportamiento.

Cuando la jerarquía se complica, considera composición en lugar de herencia. super() facilita invocar la lógica de la clase base en constructores o métodos sobrescritos.

class Animal:
    def hablar(self):
        return "..."

class Perro(Animal):
    def hablar(self):
        return "guau"

print(Perro().hablar())

Ejercicio: crea una jerarquía Figura, Circulo, Rectangulo con método area().

Mostrar solución 
import math
class Figura:
    def area(self): raise NotImplementedError
class Circulo(Figura):
    def __init__(self, r): self.r = r
    def area(self): return math.pi * self.r ** 2
class Rectangulo(Figura):
    def __init__(self, a,b): self.a,self.b=a,b
    def area(self): return self.a * self.b

Dataclasses (bonus)

Simplifican clases de datos.

Las dataclasses generan automáticamente __init__, __repr__ y comparadores. Admite valores por defecto, campos calculados y congelados (frozen=True).

from dataclasses import dataclass

@dataclass
class Punto:
    x: int
    y: int

print(Punto(1,2))
