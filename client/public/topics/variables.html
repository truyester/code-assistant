Variables

Python es de tipado dinámico. Una variable se crea al asignarle un valor.

Respeta la convención snake_case y evita reutilizar nombres para conceptos distintos. Usa type() o isinstance() solo cuando realmente necesites comprobar tipos.

mensaje = "Hola"
contador = 0
pi = 3.14159
mensaje, contador = mensaje + " mundo", contador + 1
print(mensaje, contador)

Ejercicio: intercambia los valores de dos variables sin usar una variable temporal.

Mostrar solución 
a, b = 1, 2
a, b = b, a
print(a, b)

Operadores aritméticos

Suma, resta, multiplicación, división, módulo, potencia, división entera: + - * / % ** //.

Recuerda que // trunca hacia abajo y % conserva el signo del divisor. Usa math para operaciones avanzadas como raíz cuadrada o factorial.

a, b = 7, 3
print(a + b, a - b, a * b, a / b)
print(a % b, a ** b, a // b)

Ejercicio: calcula el IMC (índice de masa corporal) con peso y altura dados.

Mostrar solución 
peso = float(input("kg: "))
altura = float(input("m: "))
imc = peso / (altura ** 2)
print(f"IMC: {imc:.2f}")

Comparación y lógicos

Comparación: == != < <= > >=. Lógicos: and, or, not.

Los operadores lógicos devuelven el operando evaluado, por lo que puedes encadenarlos en expresiones de asignación. Evita comparar booleanos con True/False; usa directamente la condición.

x, y = 10, 20
print(x < y and y < 30)   # True
print(not (x == y))       # True

Ejercicio: dada una edad y si tiene permiso, decide si puede entrar a un evento (usa and/or).

Mostrar solución 
edad = int(input("edad: "))
permiso = input("permiso (s/n): ").lower() == "s"
puede = (edad >= 18) or (edad >= 16 and permiso)
print("puede" if puede else "no puede")

Asignación y precedencia

Compuestos: +=, -=, *=, /=. Revisa precedencia.

Combinar operadores compuestos ayuda a escribir código más expresivo y evita errores al escribir la variable dos veces. Cuando la precedencia no sea obvia, usa paréntesis para dejar clara la intención.

x = 5
x *= 2 + 3   # x = 5 * (2 + 3) = 25

Ejercicio: reescribe una expresión con paréntesis para que produzca un resultado específico.

Mostrar solución 
# Ejemplo: obtener 14 a partir de 2 + 3 * 4
