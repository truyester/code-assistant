Tipos de Datos en Python

Números

Python soporta varios tipos numéricos como enteros (int), flotantes (float) y números complejos (complex).

Cadenas de texto

Las cadenas (str) permiten almacenar y manipular texto.

Listas

Las listas (list) son estructuras de datos ordenadas y mutables.

Tuplas

Las tuplas (tuple) son similares a las listas, pero son inmutables.

Diccionarios

Los diccionarios (dict) almacenan pares clave-valor.
