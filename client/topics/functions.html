Definición y retorno

Funciones se crean con def y pueden devolver valores con return.

Procura que cada función haga una sola cosa y documenta los parámetros para mejorar la legibilidad. En Python, retornar múltiples valores se logra con tuplas.

def area_circulo(r):
    return 3.1416 * r ** 2

print(area_circulo(2))

Ejercicio: define es_primo(n) que devuelva True/False.

Mostrar solución 
def es_primo(n: int) -> bool:
    if n < 2: return False
    i = 2
    while i * i <= n:
        if n % i == 0: return False
        i += 1
    return True

Parámetros y valores por defecto

Los parámetros pueden tener valores por defecto y ser nombrados.

Los argumentos por palabra clave permiten invocar funciones en cualquier orden y hacen más expresivo el código. Usa valores inmutables como default para evitar efectos laterales.

def saludar(nombre, sep=": "):
    print("Hola" + sep + nombre)

saludar("Ana")
saludar("Luis", sep=" - ")

Ejercicio: crea una función que formatee nombre y apellido con un separador configurable.

Mostrar solución 
def formato_nombre(nombre, apellido, sep=" "):
    return f"{nombre}{sep}{apellido}"

*args y **kwargs

Permiten aceptar una cantidad variable de argumentos posicionales y nombrados.

Úsalos para construir APIs flexibles (por ejemplo, envoltorios o decoradores). Documenta qué tipo de valores esperas en *args/**kwargs para evitar confusiones.

def sumar(*nums):
    return sum(nums)

print(sumar(1,2,3))

def configurar(**opciones):
    return opciones

print(configurar(color="azul", modo=True))

Ejercicio: implementa media(*nums) que ignore argumentos no numéricos.

Mostrar solución 
def media(*nums):
    vals = [n for n in nums if isinstance(n, (int,float))]
    return sum(vals)/len(vals) if vals else 0.0

Docstrings y anotaciones

Documenta y anota tipos para mejorar la legibilidad.

Las anotaciones son opcionales pero se integran bien con herramientas de análisis estático (mypy, pyright). Los docstrings siguen la convención de triple comilla y pueden describir parámetros y retornos.

def distancia(x: float, y: float) -> float:
    """Devuelve |x - y|."""
    return abs(x - y)

print(distancia.__doc__)

Ejercicio: añade docstring y anotaciones a una función propia.

Mostrar solución 
def normalizar(texto: str) -> str:
    """Recorta espacios y pasa a minúsculas."""
    return texto.strip().lower()
